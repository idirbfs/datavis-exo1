Jeux vidéos en réseau : Top 50 des jeux vidéos les plus vendus entre 1984 et 2014/h5>
Force Directed Tree
  • Explication

    Description : Cette visualisation met en réseau les 50 jeux les plus vendus par année, en intégrant leur genre.
    Légende : Date > Genre > Titre du jeu
    Type de visualisation : Qualitatif, catégoriel
    Jeu de données : Kaggle, Video Game Sales, https://www.kaggle.com/datasets/benvarghese/video-game-sales, 200 lignes, Name, Year, Genre
    Méthodologie :

    1. Filtrer le jeu de données selon les éléments souhaités : Date, Genre, Titre
    2. Importer le fichier .CSV sur Rawgraphs old
    3. Création de la hiérarchie du modèle “Cluster Dendogram”
    4. Export au format .JSON
    5. Fork et personnalisation du modèle “Force-Directed Tree” sur Observable
    Observations :

    Avant les années 2000, les jeux les plus vendus sont principalement des jeux de type “plateforme”.
