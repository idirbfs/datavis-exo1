Score critique vs score utilisateur
Bar Chart
  • Description

    Comparaison entre le score critique et le score attribué par les joueurs par jeux

    Type de visualisation : Quantitatif, catégoriel
    Jeu de données : Kaggle, Video Game Sales Data 13,000 jeux sortis entre 1977 et 2020, provenant de la base de données

    Méthodologie :
    - Nettoyage des données
    - Filtrage : name, critique score et user score : diagramme à barre
    - Génération de la visualisation via Rstudio

    Observations :
    Pattern 1 : le user score est généralement inférieur au critique score. On peut noter un écart significatif pour les jeux FIFA18, FIFA19 et FIFA20 ainsi que Madden 18, Madden 19, Madden 20 ou encore Mortal Combat 11 …
